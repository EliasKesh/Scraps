MemCachier The easiest, most advanced memcache. Start with 25MB free.

  • heroku addons:add memcachier

  • code Never change your client code

    You'll never need to make a code change with MemCachier as your cache grows. Our architecture lets us add cache servers without requiring you to change your code.

  • config No servers; no daemons

    Never create an instance. Never worry about how much memory to have per machine. Never SSH to a server, reboot a server, configure anything, rotate logs, or worry about DNS. Worry about your app instead — we do the rest for you.

  • layer High availability

    Your cache is spread across many servers. When one of those servers goes down, you’ll only lose a small percentage of your cache and hence only see a small increase in cache misses.

  • linegraph Analytics dashboard

    Our analytics dashboard gives you usage insights to help you get more out of your cache.
